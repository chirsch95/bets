How the hitter K projection works
Projection
(Hitter K% × opposing starter K% / league K%) × park × expected PA. Expected PA comes from a static lineup-slot table (leadoff ≈ 4.65, #9 ≈ 3.85). v0 treats the whole game as if every PA were vs the starter — bullpen K% blending is a future improvement.
Lines
Hitter K lines are usually 0.5 or 1.5. Distribution is Poisson-like with low mean, so the same P(over) math from the pitcher model carries over.
Heads up
Hitter projections only render for batters whose lineup card is confirmed in the MLB API — typically 2–3 hours before first pitch.
